# 🤖 Line Follower — Maker Pi RP2040

Ρομπότ ακολούθησης γραμμής (line follower) με **PID controller** και μη-γραμμικό κέρδος για ομαλή συμπεριφορά στις ευθείες και επιθετική απόκριση στις κλειστές στροφές. Γραμμένο σε **CircuitPython** για την πλακέτα Maker Pi RP2040.

## ✨ Χαρακτηριστικά

- **PID έλεγχος** (Proportional – Integral – Derivative) για ομαλή και σταθερή κίνηση.
- **Μη-γραμμικό κέρδος** (`EDGE_BOOST`): ήπια αντίδραση κοντά στο κέντρο, δυνατό «δάγκωμα» στις γωνίες.
- **Δυναμική ταχύτητα**: κόβει αυτόματα στροφές και επιταχύνει στις ευθείες.
- **Ανάκτηση γραμμής**: αν χαθεί η γραμμή, στρίβει προς την τελευταία γνωστή κατεύθυνση.
- **Γραμμή τερματισμού**: σταματάει αυτόματα όταν και οι 3 αισθητήρες δουν μαύρο.
- **Κουμπιά START / STOP** για χειρισμό χωρίς υπολογιστή.

## 🔧 Υλικό (Hardware)

- Πλακέτα **Cytron Maker Pi RP2040**
- 2× μοτέρ DC (αριστερό / δεξί) σε διαμόρφωση dual-PWM H-bridge
- 3× αναλογικοί αισθητήρες γραμμής (IR reflectance)
- 2× κουμπιά (push buttons)

### Συνδεσμολογία (Pinout)

| Εξάρτημα | Pin(s) | Σημείωση |
|----------|--------|----------|
| Αριστερό μοτέρ | `GP8` (M1A) · `GP9` (M1B) | dual-PWM H-bridge |
| Δεξί μοτέρ | `GP10` (M2A) · `GP11` (M2B) | dual-PWM H-bridge |
| Αισθητήρας αριστερά | `GP26` (A0) | αναλογικός |
| Αισθητήρας κέντρο | `GP27` (A1) | αναλογικός |
| Αισθητήρας δεξιά | `GP28` (A2) | αναλογικός |
| Κουμπί START | `GP20` | active LOW (pull-up) |
| Κουμπί STOP | `GP21` | active LOW (pull-up) |

## 📦 Εγκατάσταση

1. Εγκατέστησε το **CircuitPython** στην πλακέτα Maker Pi RP2040.
   Κατέβασέ το από το [circuitpython.org](https://circuitpython.org/board/cytron_maker_pi_rp2040/) και αντίγραψε το αρχείο `.uf2` στη συσκευή σε λειτουργία bootloader (κράτα πατημένο το BOOTSEL ενώ συνδέεις το USB).
2. Αντίγραψε τον κώδικα ως `code.py` στη ρίζα του δίσκου **CIRCUITPY** που εμφανίζεται.
3. Το πρόγραμμα ξεκινάει αυτόματα μόλις αντιγραφεί.

> Οι βιβλιοθήκες `board`, `pwmio`, `digitalio`, `analogio` και `time` είναι ενσωματωμένες στο CircuitPython — δεν χρειάζεται extra εγκατάσταση.

## ▶️ Χρήση

1. Τοποθέτησε το ρομπότ πάνω στη γραμμή.
2. Πάτα το κουμπί **GP20 (START)** για εκκίνηση.
3. Πάτα το κουμπί **GP21 (STOP)** για σταμάτημα οποιαδήποτε στιγμή.
4. Το ρομπότ σταματάει επίσης **αυτόματα** όταν φτάσει σε γραμμή τερματισμού (μαύρο και στους 3 αισθητήρες).

Για επανεκκίνηση μετά από σταμάτημα, ξαναπάτα το **GP20**.

## 🎚️ Βαθμονόμηση αισθητήρων (Calibration)

Οι τιμές των αισθητήρων κανονικοποιούνται στο εύρος `0.0` (λευκό) – `1.0` (μαύρη γραμμή) με βάση τις σταθερές:

```python
CAL_MIN = [2896, 20148, 3440]    # τιμές πάνω σε ΛΕΥΚΟ
CAL_MAX = [55325, 62015, 60974]  # τιμές πάνω σε ΜΑΥΡΟ
```

**Πρέπει να τις προσαρμόσεις στη δική σου πίστα και φωτισμό.** Για να βρεις τις σωστές τιμές, διάβασε τις ακατέργαστες τιμές (`sensor_left.value` κ.λπ.) πάνω σε καθαρό λευκό και πάνω σε μαύρο, και πέρασέ τες αντίστοιχα στο `CAL_MIN` και `CAL_MAX` (σειρά: αριστερά, κέντρο, δεξιά).

## ⚙️ Ρύθμιση PID (Tuning)

Όλες οι παράμετροι βρίσκονται μαζεμένες στην ενότητα `ΠΑΡΑΜΕΤΡΟΙ PID` του κώδικα.

| Παράμετρος | Προεπιλογή | Τι κάνει |
|------------|-----------|----------|
| `BASE_SPEED` | `55` | Ταχύτητα στην ευθεία |
| `TURN_SPEED` | `22` | Ταχύτητα μέσα στη στροφή |
| `MAX_SPEED` | `80` | Μέγιστη ταχύτητα μοτέρ |
| `MIN_SPEED` | `-55` | Επιτρέπει pivot (αντίστροφη περιστροφή εσωτερικού τροχού) |
| `Kp` | `48.0` | Άμεση διόρθωση (analog του «πόσο δυνατά αντιδρά») |
| `Ki` | `0.4` | Διόρθωση σταθερού σφάλματος |
| `Kd` | `20.0` | Απόσβεση ταλαντώσεων |
| `EDGE_BOOST` | `1.5` | Μη-γραμμικό κέρδος — επιθετικότητα στις γωνίες |
| `STOP_THRESHOLD` | `0.7` | Όριο πάνω από το οποίο ένας αισθητήρας θεωρείται «μαύρο» |
| `STOP_CONFIRM` | `3` | Συνεχόμενες αναγνώσεις για επιβεβαίωση γραμμής τερματισμού |

### Συχνά προβλήματα

**Τρεμοπαίζει / ζιγκ-ζαγκ στην ευθεία:**
- Κατέβασε το `Kp` (π.χ. 48 → 40 → 35) — αυτό είναι ο βασικός ένοχος.
- Αν μετά αργεί στις στροφές, ανέβασε το `Kd` για απόσβεση (20 → 26 → 32).
- Κατέβασε το `EDGE_BOOST` (π.χ. 1.0) για πιο ήπια αντίδραση κοντά στο κέντρο.
- Άλλαζε **μία τιμή τη φορά**.

**Κόβει αργά τις στροφές / βγαίνει εκτός:**
- Ανέβασε το `Kp` ή το `EDGE_BOOST`.
- Χαμήλωσε το `TURN_SPEED`.

**Σταματάει κατά λάθος σε κλειστές στροφές:**
- Ανέβασε το `STOP_CONFIRM` (π.χ. 5) ή το `STOP_THRESHOLD` (π.χ. 0.8).

**Δεν πιάνει τη γραμμή τερματισμού:**
- Κατέβασε το `STOP_THRESHOLD` και βεβαιώσου ότι η μπάρα τερματισμού είναι αρκετά φαρδιά ώστε να την δουν και οι 3 αισθητήρες ταυτόχρονα.

## 🧠 Πώς δουλεύει

1. Οι 3 αισθητήρες διαβάζονται και κανονικοποιούνται (`read_normalized`).
2. Υπολογίζεται η **θέση της γραμμής** ως σταθμισμένος μέσος όρος (`line_position`): `-1.0` = τελείως αριστερά, `0.0` = κέντρο, `+1.0` = τελείως δεξιά.
3. Το **PID** μετατρέπει το σφάλμα θέσης σε διόρθωση, με μη-γραμμική ενίσχυση στα άκρα.
4. Η διόρθωση εφαρμόζεται διαφορικά στους δύο τροχούς, ενώ η βασική ταχύτητα μειώνεται δυναμικά όσο μεγαλώνει το σφάλμα.
5. Αν χαθεί η γραμμή, το ρομπότ κάνει pivot προς την τελευταία γνωστή κατεύθυνση για έως `LOST_TIMEOUT` δευτερόλεπτα.
